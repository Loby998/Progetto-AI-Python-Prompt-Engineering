📦 INSTALLAZIONE DEL PROGETTO – VERSIONE OLLAMA

1️⃣ Installa Python
- Vai su https://python.org/downloads
- Scarica l’ultima versione (Python 3.x)
- Durante l’installazione spunta “Add Python to PATH”
- Verifica l’installazione aprendo il terminale e scrivendo:
  python --version

2️⃣ Installa Visual Studio Code
- Vai su https://code.visualstudio.com
- Installa normalmente
- Apri VS Code → premi Ctrl+Shift+X → cerca “Python” → installa l’estensione ufficiale Microsoft

3️⃣ Installa Ollama
- Vai su https://ollama.com
- Scarica e installa
- Apri il terminale e installa il modello necessario al progetto:
  ollama pull qwen2.5:1.5b-instruct

4️⃣ Installa la libreria Python richiesta dal progetto
Apri il terminale (o quello integrato in VS Code) e scrivi:
  pip install requests

5️⃣ Avvio del progetto
- Apri VS Code
- Apri la cartella del progetto
- Apri il file: valutatore_risposte.py
- Assicurati che Ollama sia avviato (si avvia automaticamente)
- Esegui il programma con:
  python valutatore_risposte.py

Il sistema ora è pronto per generare 5 risposte, valutarle e scegliere la migliore.
