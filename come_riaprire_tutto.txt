1. Installa Python

Vai su python.org/downloads
Scarica l'ultima versione (3.x)
Durante l'installazione spunta "Add Python to PATH" ⚠️ importante!
Verifica che funzioni aprendo il terminale e scrivendo:
python --version
Chiudi.

2. Installa VS Code
- Vai su **code.visualstudio.com**
- Scarica e installa normalmente
- Apri VS Code e installa l'estensione **Python** (di Microsoft):
- Ctrl+Shift+X → cerca "Python" → installa

3. Installa LM Studio
- Vai su **lmstudio.ai**
- Scarica e installa normalmente
- Apri LM Studio
- Vai nella sezione **Discover** (icona 🔍) e cerca:
  qwen2.5 1.5b instruct

Scaricalo (circa 1,46 GB) -> quello più downloadato


4. Installa la libreria requests
Apri il terminale (anche quello di VS Code) e scrivi:
pip install requests

5. Ogni volta che vuoi usare il programma
Step 1 — Apri LM Studio:

Vai su Local Server (icona <-> a sinistra)
Seleziona il modello Qwen2.5 e caricalo
Premi Start Server

Step 2 — Apri VS Code:
Apri il file validatore_risposte.py dentro la cartella zippata allegata alla mail
Apri il terminale di vs code e scrivi:
python valutatore_risposte.py
